# Dah Settle - Support

## Frequently Asked Questions

### How do I add a new expense?
Tap the "+" button on the main screen, enter the expense details, and choose how to split it between members.

### How do I create a new space?
Open the menu, tap "Manage Spaces", then tap "New Space" to create a separate household or group.

### How do I set up recurring bills?
Go to Menu → Recurring Expenses → Add a recurring bill. It will automatically add each month on the day you specify.

### Can I backup my data?
Yes! Go to Menu → Settings → Export Data to save a backup file. You can import it later to restore your data.

### Where is my data stored?
All data is stored locally on your device. Nothing is uploaded to the cloud.

### How do I switch between spaces?
Open the menu and tap on any space in the "Switch Space" section.

### How do I mark a bill as paid?
Tap on any expense in the main list and toggle the "Paid" switch.

### Can I split bills unevenly?
Yes! When adding an expense, choose "Custom Percentages" to set different split amounts for each person.
